# OldPeopleCure

A React Native mobile application for elderly care, built with Expo.

## Quick Start

1. **Navigate to the project folder:**
   ```bash
   cd OldPeopleCure
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch the app:**
   ```bash
   npm start
   ```

Scan the QR code with the **Expo Go** app (Android/iOS) to run the application on your device.
